# oracle-agent-memory

Artefacts showing how to use Oracle Agent Memory SDK with Autonomous Transaction Processing. 

The blog for this can be found on Medium.

- oracle-agent-memory.py shows you how to connect to the Oracle Agent Memory SDK
- the HiveMind folder gives a sample Engagement Tracking app that uses Oracle Agent Memory to track engagements over time for AI powered note taking
- the Skills folder packages some example skills for adding, updating or deleting memories for HiveMind using Cline or Codex


NOTE: this is a sample app to demonstrate use of the Oracle Agent Memory. As such it should be treated as a demo only for now. Use at your own discretion.
